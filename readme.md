# WhatsTing - Aplikasi Pesan & Sosial Media Serba Bisa

Aplikasi mobile berbasis Flutter yang mengintegrasikan chat, status, reels, marketplace, dan AI assistant.

## Fitur Utama
- 💬 Chat & Pesan Instan dengan bubble
- 🔵 Status Stories seperti Instagram
- 🎬 Reels Video Pendek seperti TikTok
- 🛒 Marketplace dengan keranjang belanja
- 🤖 Wit AI Assistant
